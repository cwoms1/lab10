Understanding XML and JavaScript Basics

What is XML?

XML (eXtensible Markup Language) is a markup language that defines rules for encoding documents in a format that is both human-readable and machine-readable.

How is XML used?

XML is commonly used for representing structured data. Here's a basic example of an XML document representing information about a person:

            
                <person>
                    <name>John Doe</name>
                    <age>30</age>
                    <city>Example City</city>
                </person>

What does XML look like? (Example)

            
                <book>
                    <title>Introduction to XML</title>
                    <author>Jane Smith</author>
                    <published>2022</published>
                </book>

What is JavaScript?

JavaScript is a scripting language that allows you to implement complex features on web pages. It adds interactivity and enhances the user experience.

How is JavaScript related to HTML and CSS?

JavaScript can interact with HTML and CSS to dynamically modify content and styles. Here's a basic example:

            
                <button onclick="changeColor()">Change Color</button>

                <script>
                    function changeColor() {
                        document.body.style.backgroundColor = 'lightblue';
                    }
                </script>

Common JavaScript Elements, Properties, Methods, and Functions

Example of an element: document.getElementById('example')

Example of a property: element.style.color = 'red'

Example of a method: console.log('Hello, World!')

Example of a function: function greet(name) { return 'Hello, ' + name + '!'; }
